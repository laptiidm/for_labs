# **Звіт з лабораторної роботи №1**

## **1. Тема роботи**

Керування вмиканням та вимиканням світлодіода на Arduino за допомогою програмного коду.

---

## **2. Конкретне завдання**

### **Основне завдання:**

Написати програму, яка:

* вмикає світлодіод на **100 мс**;
* вимикає світлодіод на **900 мс**.

### **Завдання для самостійного виконання:**

1. Змінити тривалість роботи маячка:

   * світиться **0,5 с**,
   * пауза **1 с**.

2. Модифікувати код так, щоб:

   * після запуску пристрою світлодіод **світиться 3 секунди**,
   * а потім блимає у стандартному режимі (100/900 мс).

---

## **3. Код програми**

### **3.1. Базовий код (100/900 мс)**

```cpp
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(100);
  digitalWrite(13, LOW);
  delay(900);
}
```

---

### **3.2. Самостійне завдання №1 (0.5 с / 1 с)**

```cpp
void loop() {
  digitalWrite(13, HIGH);
  delay(500); 
  digitalWrite(13, LOW);
  delay(1000);
}
```

---

### **3.3. Самостійне завдання №2 (3 секунди світить → стандартне блимання)**

```cpp
void setup() {
  pinMode(13, OUTPUT);
  digitalWrite(13, HIGH);
  delay(3000); // 3 секунди
}

void loop() {
  digitalWrite(13, HIGH);
  delay(100);
  digitalWrite(13, LOW);
  delay(900);
}
```
