## 🛠️ Чекліст: Налаштування та Запуск

### 1. Підготовка середовища

* **[ ] Встановити Node.js (версію LTS)**
* *Чому саме так:* Браузери не вміють напряму читати React-компоненти та JSX. Node.js потрібен як платформа для запуску локальних інструментів розробника (зокрема збирача Vite), які транспілюють (перекладають) твій сучасний React-код у звичайний JavaScript, зрозумілий будь-якому браузеру.


* **[ ] Перевірити встановлення в терміналі:** `node -v` та `npm -v`

### 2. Створення та ініціалізація проєкту

* **[ ] Згенерувати шаблон через Vite:**
```bash
npm create vite@latest react-lab -- --template react

```


* *Чому саме так:* Флаг `--template react` одразу створює чисту, правильну структуру папок з налаштованим конфігом для React. Це позбавляє від необхідності вручну прописувати правила збирання проєкту.


* **[ ] Перейти в директорію проєкту:** `cd react-lab`
* **[ ] Встановити пакети:** `npm install`
* *Чому саме так:* Сама команда генерації проєкту створює лише файл-маніфест `package.json` (список того, що *треба* завантажити). Команда `npm install` викачує реальний код бібліотек React та ReactDOM у папку `node_modules`.



### 3. Локальний старт

* **[ ] Запустити сервер розробки:** `npm run dev`
* *Чому саме так:* Vite запускає надшвидкий локальний вебсервер. Він активує технологію **HMR (Hot Module Replacement)** — як тільки ти зберігаєш зміни у файлі `App.jsx`, сторінка в браузері оновлюється миттєво без повного перезавантаження сайту, що економить купу часу при розробці інтерфейсу.
